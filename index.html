PassQR NFC

QR-OTP generated with every scan using a Yubikey Neo or equivalent.

PassQR NFC allows you to separate your identity keys. When accessing your online assets, you will need both your smartphone and your NFC key to generate a unique One Time Password. This QR-OTP will then be presented to a companion desktop or mobile app to be validated for a secure authenticated session.

We are currently compatible with Yubico Yubikey Neo. Generated QR-OTP can be validated on Yubikey's OTP demo validation site.

PassQR TouchID

QR-OTP generated with each TouchID scan.

This is a fully software solution to generate a QR-OTP. Once your fingerprint is scanned and validated to be correct by the smartdevice, a QR-OTP will be generated. This QR-OTP will then be presented to a companion desktop or mobile app to be validated for a secure authenticated session.

With PassQR TouchID, this can be used as a two factor authentication for access to websites and enterprise applications.

PassQR Chrome Extension

QR-OTP can be scanned and validated with PassQR Chrome Extension

Works with both PassQR NFC as well as PassQR TouchID, PassQR Chrome Extension can be installed from the Chrome WebStore. Once installed it can be used to scan QR-OTPs and validated with respective backend services to ensure secure, roundtrip authentication.

PassQR Chrome Extension is an app developed to showcase QR-OTP Passwordless technique that effectively bypasses keyloggers vulnerabilities.
